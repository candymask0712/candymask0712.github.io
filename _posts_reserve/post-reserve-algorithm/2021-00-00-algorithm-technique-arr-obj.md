### **1. 배열**

- slice

  ```javascript
  let A = arr.slice(st, end); // st 이상 end '미만' 자름 / 원배열 수정 X (할당 필요)
  let B = arr.slice(n); // n번째 요소부터(n포함) 마지막 요소까지 잘라냄
  ```

- Array

  ```javascript
  let A = Array(n).fill(x); // x로 n만큼 채워진 행렬
  console.log(Array(3).fill(0)); // [0,0,0]
  ```
