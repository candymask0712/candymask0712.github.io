### **기억해야할 falsy 값 6가지**

- 다음 falsy 값은 조건절에 들어가면 문이 실행되지 않음
- 조건부실행이 필요한 구문이나 알고리즘 풀이 유용할듯 함

```javascript
if (false)
if (null)
if (undefined)
if (0)
if (NaN)
if ('')
```
